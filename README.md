# Perlin noise map

## About

[![Screen Shot][screenshot]]()

An infinite terrain procedurally generated with Perlin noise.

### Built With

* [raylib-go](https://github.com/gen2brain/raylib-go)

## Getting Started

### Running

    make run

### Building

    make build

### Controls

- Navigate: QWSD/ZQSD + Mouse
- Show FPS: F1
